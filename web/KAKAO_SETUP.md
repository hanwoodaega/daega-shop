# 카카오톡 공유 설정 가이드

## 카카오톡 공유가 실패하는 경우 확인사항

### 1. 카카오톡 개발자 콘솔 설정

1. **https://developers.kakao.com** 접속
2. **내 애플리케이션** → 앱 선택
3. **앱 키** → **JavaScript 키** 복사하여 `.env.local`에 설정:
   ```env
   NEXT_PUBLIC_KAKAO_APP_KEY=여기에_JavaScript_키_입력
   ```

### 2. 플랫폼 등록 (중요!)

카카오톡 공유가 작동하려면 **웹 플랫폼**을 등록해야 합니다:

1. **내 애플리케이션** → 앱 선택
2. **앱 설정** → **플랫폼** 탭
3. **Web 플랫폼 등록** 클릭
4. **사이트 도메인** 입력:
   - **PC에서 테스트**: `http://localhost:3000`
   - **모바일에서 테스트**: `http://[컴퓨터IP주소]:3000` (예: `http://192.168.0.10:3000`)
   - **프로덕션**: 실제 도메인 (예: `https://daega-shop.vercel.app`)
   
   **중요**: 모바일에서 테스트하려면 컴퓨터의 IP 주소를 사용해야 합니다!
   
   - 여러 도메인을 등록하려면 각각 따로 등록하세요
   - 또는 IP 주소만 등록해도 PC와 모바일 모두에서 작동할 수 있습니다 (같은 네트워크에서)
   
5. **저장** 클릭

#### 컴퓨터 IP 주소 확인 방법:

**Windows:**
```cmd
ipconfig
```
IPv4 주소를 찾으세요 (예: `192.168.0.10`)

**Mac/Linux:**
```bash
ifconfig
# 또는
ip addr
```

**모바일에서 접속:**
- PC와 모바일이 같은 Wi-Fi 네트워크에 연결되어 있어야 합니다
- 모바일 브라우저에서 `http://[컴퓨터IP]:3000`으로 접속

### 3. 카카오톡 로그인 활성화 (선택사항)

공유만 사용하는 경우 필수는 아니지만, 일부 기능을 위해 필요할 수 있습니다:

1. **제품 설정** → **카카오 로그인** 활성화
2. **Redirect URI** 등록:
   - 개발: `http://localhost:3000`
   - 프로덕션: 실제 도메인

### 4. 이미지 URL 확인

카카오톡 공유에 사용되는 이미지는 다음 조건을 만족해야 합니다:

- ✅ HTTPS로 접근 가능해야 함
- ✅ CORS 설정이 되어 있어야 함
- ✅ 이미지 크기: 최소 200x200px (권장: 800x800px 이상)
- ✅ 이미지 형식: JPG, PNG, GIF

### 5. 일반적인 에러 해결

#### "요청 실패" 에러
- **원인**: 플랫폼이 등록되지 않았거나 도메인이 일치하지 않음
- **해결**: 위의 "플랫폼 등록" 단계를 확인하세요

#### "앱 키가 설정되지 않았습니다" 에러
- **원인**: `.env.local` 파일에 `NEXT_PUBLIC_KAKAO_APP_KEY`가 없음
- **해결**: 환경 변수를 설정하고 개발 서버를 재시작하세요

#### "이미지 로드 실패" 에러
- **원인**: 이미지 URL이 접근 불가능하거나 CORS 문제
- **해결**: 이미지가 HTTPS로 접근 가능한지 확인하고, Supabase Storage의 CORS 설정을 확인하세요

### 6. 테스트 방법

1. 개발 서버 재시작:
   ```bash
   npm run dev
   ```

2. 브라우저 콘솔에서 확인:
   - 카카오톡 SDK 로드 여부
   - SDK 초기화 여부
   - 공유 요청 데이터

3. 카카오톡 개발자 콘솔 → **제품 설정** → **카카오톡 채널** 연결 확인

### 7. 프로덕션 배포 시

Vercel 등에 배포할 때는 환경 변수를 설정해야 합니다:

1. Vercel 대시보드 → 프로젝트 선택
2. **Settings** → **Environment Variables**
3. `NEXT_PUBLIC_KAKAO_APP_KEY` 추가
4. **플랫폼**에 프로덕션 도메인 등록
