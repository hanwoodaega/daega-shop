# 선물 카드 디자인 이미지

이 폴더에 선물 카드 디자인 이미지를 추가하세요.

## 필요한 이미지 파일

다음 4개의 이미지 파일을 추가해주세요:

1. `birthday.jpg` - 생일 축하 카드 디자인
2. `anniversary.jpg` - 기념일 카드 디자인  
3. `thanks.jpg` - 감사 인사 카드 디자인
4. `custom.jpg` - 커스텀 카드 디자인

## 이미지 권장 사양

- **비율**: 4:3 (가로:세로)
- **해상도**: 최소 800x600px 이상 권장
- **형식**: JPG 또는 PNG
- **크기**: 각 파일 500KB 이하 권장

## 메시지 오버레이 영역

카드 이미지의 중앙 영역에 메시지가 오버레이됩니다. 
디자인 시 중앙 부분에 텍스트가 들어갈 공간을 고려해주세요.
